Usage

Parse a local file and get structure in memory:

data = Fallout2Dat.from_file("path/to/local/file.fallout2_dat")

Or parse structure from a string of bytes:

bytes = "\x00\x01\x02..."
data = Fallout2Dat.new(Kaitai::Struct::Stream.new(bytes))

After that, one can get various attributes from the structure by invoking getter methods like:

data.footer # => get footer

    def contents

        _pos = io.pos
        io.seek(offset)

  def footer

    @_io.seek((_io.size - 8))

    @_io.seek(_pos)
    @index
