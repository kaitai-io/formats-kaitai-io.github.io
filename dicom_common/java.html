Usage

Parse a local file and get structure in memory:

DicomCommon data = DicomCommon.fromFile("path/to/local/file.dcm");

Or parse structure from a byte array:

byte[] someArray = new byte[] { ... };
DicomCommon data = new DicomCommon(new KaitaiStream(someArray));
