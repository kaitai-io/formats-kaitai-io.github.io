Usage

Parse a local file and get structure in memory:

AndesFirmware data = AndesFirmware.fromFile("path/to/local/file.Andes Firmware Image layout as seen in MT76 Wifi Chipsets");

Or parse structure from a byte array:

byte[] someArray = new byte[] { ... };
AndesFirmware data = new AndesFirmware(new ByteBufferKaitaiStream(someArray));

After that, one can get various attributes from the structure by invoking getter methods like:

data.imageHeader() // => get image header
