.mp3 file format: C# parsing library

File extension

["mp3"]

KS implementation details

License: CC0-1.0

This page hosts a formal specification of .mp3 file format using Kaitai Struct. This specification can be automatically translated into a variety of programming languages to get a parsing library.

Usage

Parse a local file and get structure in memory:

var data = Id3v23.FromFile("path/to/local/file.["mp3"]");

Or parse structure from a byte array:

byte[] someArray = new byte[] { ... };
var data = new Id3v23(new KaitaiStream(someArray));

After that, one can get various attributes from the structure by accessing properties like:

data.Tag // => get tag
