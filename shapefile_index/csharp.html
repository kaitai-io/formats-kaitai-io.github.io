Shapefile index file: C# parsing library

File extension

Usage

Parse a local file and get structure in memory:

var data = ShapefileIndex.FromFile("path/to/local/file.shx");

Or parse structure from a byte array:

byte[] someArray = new byte[] { ... };
var data = new ShapefileIndex(new KaitaiStream(someArray));

After that, one can get various attributes from the structure by accessing properties like:

data.Records // => the size of this section of the file in bytes must equal (header.file_length * 2) - 100
