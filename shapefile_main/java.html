Usage

Parse a local file and get structure in memory:

ShapefileMain data = ShapefileMain.fromFile("path/to/local/file.shp");

Or parse structure from a byte array:

byte[] someArray = new byte[] { ... };
ShapefileMain data = new ShapefileMain(new ByteBufferKaitaiStream(someArray));

After that, one can get various attributes from the structure by invoking getter methods like:

data.records() // => the size of this section of the file in bytes must equal (header.file_length * 2) - 100
