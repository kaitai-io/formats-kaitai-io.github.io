AUTOSAR SOME/IP Service Discovery Options: Java parsing library

FormatOptions are used to transport additional information to the entries. This includes forinstance the information how a service instance is reachable (IP-Address, TransportProtocol, Port Number).

Usage

Parse a local file and get structure in memory:

SomeIpSdOptions data = SomeIpSdOptions.fromFile("path/to/local/file.AUTOSAR SOME/IP Service Discovery Options");

Or parse structure from a byte array:

byte[] someArray = new byte[] { ... };
SomeIpSdOptions data = new SomeIpSdOptions(new ByteBufferKaitaiStream(someArray));

After that, one can get various attributes from the structure by invoking getter methods like:

data.entries() // => get entries
