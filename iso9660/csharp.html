.iso file format: C# parsing library

File extension

iso

This page hosts a formal specification of .iso file format using Kaitai Struct. This specification can be automatically translated into a variety of programming languages to get a parsing library.

  • Overview
  • C++/STL
  • C#
  • GraphViz

Usage

Parse a local file and get structure in memory:

var data = Iso9660.FromFile("path/to/local/file.iso");

Or parse structure from a byte array:

byte[] someArray = new byte[] { ... };
var data = new Iso9660(new KaitaiStream(someArray));

After that, one can get various attributes from the structure by accessing properties like:

data.SectorSize // => get sector size
